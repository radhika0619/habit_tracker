# Habit Tracker

A full-stack habit tracking web application built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features

* Add, edit, and delete habits
* Habit streak tracking
* Progress bars
* Weekly progress visualization
* Achievement badges
* Motivational quotes
* Dark/Light mode
* User authentication
* MongoDB database storage

## Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose
